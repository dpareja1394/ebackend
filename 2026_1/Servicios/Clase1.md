# Desarrollo de Sistemas Basados en Servicios
## Descripción del Curso

En este curso trabajaremos con **microservicios** utilizando dos lenguajes de programación:

- **Java** con **Spring Boot** como framework principal para desarrollar microservicios
- **Go** para servicios adicionales de lectura y escritura

### Tecnologías Principales

- **Java 17+** con Spring Boot
- **Go 1.18+**
- **PostgreSQL** como base de datos relacional
- Herramientas de desarrollo y gestión de dependencias

### Instalación y Configuración

#### Paso 1: Instalación de Java
1. Descargar JDK 17+ desde [oracle.com](https://www.oracle.com/java/technologies/downloads/)
2. Configurar variables de entorno (JAVA_HOME)
3. Verificar: `java -version`

#### Paso 2: Instalación de Spring Boot
1. Instalar Maven o Gradle
2. Crear proyecto con Spring Boot CLI o Maven archetype
3. Configurar `pom.xml` con dependencias necesarias

#### Paso 3: Instalación de Go
1. Descargar desde [golang.org](https://golang.org/dl/)
2. Configurar variables de entorno (GOPATH, GOROOT)
3. Verificar: `go version`

#### Paso 4: Instalación de PostgreSQL
1. Descargar desde [postgresql.org](https://www.postgresql.org/download/)
2. Configurar usuario y contraseña
3. Crear base de datos para el proyecto
4. Verificar conexión con herramientas como pgAdmin

### Objetivos del Semestre

Desarrollar una arquitectura de microservicios resiliente, escalable y mantenible.
